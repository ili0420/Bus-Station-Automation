Yukarıda bulunan resimde admin mailini ve şifresini girerek Admin sayfasına yönlendirilir 
![image](https://github.com/ili0420/Bus-Station-Automation/assets/135699517/0e6edf68-91fe-4918-a40b-41d5fd741a73)
